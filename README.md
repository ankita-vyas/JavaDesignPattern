# Java Design Pattern Examples

This repository demonstrates core Java design patterns organized by category.

## Included Patterns

- Creational
  - Singleton
  - Factory
  - Builder
  - Prototype
- Structural
  - Adapter
  - Decorator
  - Facade
  - Proxy
- Behavioral
  - Strategy
  - Observer
  - Command
  - Iterator

## Run

Use Maven to compile and run the demo:

```bash
mvn compile exec:java -Dexec.mainClass="com.javadp.DesignPatternsDemo"
```
